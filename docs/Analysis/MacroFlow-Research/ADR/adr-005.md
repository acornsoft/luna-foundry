# ADR 005: Add Document Generator Skill for Branded Deliverables

**Date**: 2026-01-03  
**Status**: Accepted  
**Context**: Client deliverables require professional, branded documents in multiple formats. Existing processes lack integrated conversion with template management and Mermaid processing.  
**Decision**: Implement the Document Generator as a Skill in `src/foundry/skills/document-generator/`, incorporating PowerShell scripts, JSON configs, and support for client template overrides.  
**Consequences**:  

- Positive: Enables ad-hoc branded document generation; supports MacroFlow Implement phase; flexible template system.  
- Negative: Depends on Pandoc and external tools; requires template setup for customization.
